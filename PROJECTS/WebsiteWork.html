Interactive Web Game

Another project I made was an interactive web game. This was set for a younger audience and was another project in university. Now expanding from basic HTML towards JavaScript and Jquery specifically.

Definitely a challenge, but I learnt a lot about responsiveness, adding sound, animations and game features like drag and drop. On top of this I also created every single individual asset that you can see, creating it all in illustrator mainly, with a few decorative additional features from Photoshop.

The project really helped me develop skills for creating both interactive code and vector art. Refining things with features like gradients, drop shadows and bevelling to give the game its unique visual.
