Posters

In these posters, a minimalistic design highlights the "Let's Talk Tash" campaign. Flat, stylized illustrations with subtle shading and texture add depth while maintaining a clean look.

Each poster uses a distinct muted background (burgundy, orange, teal) to create contrast and consistency. Bold typography pairs with "TASH" in a distressed texture, adding visual interest.

Techniques like color contrast, texture overlays, and simple illustrations enhance the message. Created in Photoshop, the posters combine modern design with a playful, impactful tone.
